Data Watchtower

Welcome to Data Watchtower! This project shines a light on how AI companies handle your data, breaking down data policies, security measures, and user controls in an easy-to-compare format. It's just getting started and not comprehensive yet - but that's where you come in! If you're curious about other AI companies or passionate about data transparency, why not join? Check out the GitHub repo and help make AI a bit more see-through!
